Commands

Player 1

  • Go right: "f" key
  • Go left: "s" key
  • Jump: "e" key
  • Go down: "d" key
  • shot: "space" key

Player 2

  • Go right: "right arrow" key
  • Go left: "left arrow" key
  • Jump: "up arrow" key
  • Go down: "down arrow" key
  • shot: "0" key
